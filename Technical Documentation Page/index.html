Introduction

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes". It has imperative, object-oriented, and generic programming features, while also providing facilities for low-level memory manipulation. C++ is widely used for systems and application software, game development, drivers, client-server applications, and embedded firmware.

What you should already know

This guide assumes you have the following basic background:

  • Basic familiarity with using computers.
  • A general understanding of programming concepts.
  • A basic understanding of creating and managing files.
Hello world

To get started with writing C++, open your code editor and write the following C++ code:

#include <iostream> int main() {     std::cout << "Hello, World!" << std::endl;     return 0; }

Save the file on your computer. Name the file "main.cpp" and compile and run it using your C++ compiler.

Anatomy of a C++ program

Let's further explore our Hello World program from the previous example. The anatomy of our program is:

  • The preprocessor directive: This includes the #include <iostream> statement which tells the preprocessor to include the iostream standard library before actual compilation starts.
  • The main function: This is the entry point of every C++ program. The execution of code begins from the main function.
  • The output statement: The std::cout << "Hello, World!" << std::endl; statement prints the text "Hello, World!" to the console.
  • The return statement: The return 0; statement ends the main function and returns 0 to the calling process.
Variables and Types

In C++, variables are used to store data. Each variable must have a type, which defines the kind of data it can hold. Common types include:

  • int: Integer type.
  • float: Floating-point type.
  • double: Double-precision floating-point type.
  • char: Character type.
  • bool: Boolean type.

Here is an example of variable declarations:

int myInt = 5; float myFloat = 3.14; double myDouble = 2.71828; char myChar = 'A'; bool myBool = true;
Control Flow

Control flow statements are used to determine the flow of execution of a program. Common control flow statements include:

  • if, else: Conditional statements.
  • for, while, do-while: Looping statements.
  • switch: Multi-way branch statement.

Here is an example of an if-else statement:

int number = 10; if (number > 0) {     std::cout << "Positive number" << std::endl; } else {     std::cout << "Negative number" << std::endl; }
Functions

Functions are blocks of code that perform specific tasks. They help in modularizing the code and reusing it. Here is an example of a simple function:

int add(int a, int b) {     return a + b; } int main() {     int sum = add(5, 3);     std::cout << "Sum: " << sum << std::endl;     return 0; }
Classes and Objects

C++ supports object-oriented programming. Classes are blueprints for creating objects. Here is an example of a simple class:

class Car { public:     string brand;     int year;     void display() {         std::cout << "Brand: " << brand << ", Year: " << year << std::endl;     } }; int main() {     Car myCar;     myCar.brand = "Toyota";     myCar.year = 2020;     myCar.display();     return 0; }
Pointers and References

Pointers and references are used to manipulate data stored in memory. Here is an example of using pointers:

int main() {     int var = 20;     int* ptr = &var;     std::cout << "Value of var: " << var << std::endl;     std::cout << "Value of *ptr: " << *ptr << std::endl;     return 0; }
Standard Template Library

The Standard Template Library (STL) is a powerful set of C++ template classes to provide general-purpose classes and functions with templates that implement many popular and commonly used algorithms and data structures like vectors, lists, queues, and stacks.

#include <vector> int main() {     std::vector<int> v = {1, 2, 3, 4, 5};     for(int i : v) {         std::cout << i << ' ';     }     return 0; }
Input/Output

C++ uses streams for input and output operations. Here is an example of basic input and output:

#include <iostream> int main() {     int age;     std::cout << "Enter your age: ";     std::cin >> age;     std::cout << "Your age is: " << age << std::endl;     return 0; }
